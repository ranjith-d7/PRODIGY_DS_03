# PRODIGY_DS_03
TASK-3 FOR PRODIGY INFOTECH DATA SCIENCE INERNSHIP

Customer Purchase Prediction Using Decision Trees

Description: This project implements a decision tree classifier to predict whether a customer will purchase a product or service based on their demographic and behavioral data. Dataset: Bank Marketing dataset from the UCI Machine Learning Repository he classification goal is to predict if the client will subscribe (yes/no) a term deposit (variable y).
